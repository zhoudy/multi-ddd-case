本规范参考于蚂蚁工程规范，主要目的用户规范DDD规范下的工程目录及包结构。

对于showcase-service中包含比较多的领域实体、值对象、聚合、领域服务、资源库、基础设施等，为了控制模块不要过多，
同时考虑到一般性项目的体量，做了适当裁剪，当然您可以对本工程在进行模块化拆解，如果你愿意。
